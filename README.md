# spotify-imersao-alura-2024
Construindo um Streaming de música

## RUN
1. Instale o NodeJS no seu pc

2. Abra o prompt de comando (CMD) e baixe o pacote JSON SERVER com o código abaixo:
npm i json-server -g

3. Abra o prompt de comando e utilize o código abaixo para simular uma API:
json-serve --watch caminho_do_arquivo --port 3000

4. Agora é só abrir o projeto com o plugin chamado Live Server do editor de texto chamado Visual Studio Code (VS Code) e testar seu projeto ^^
